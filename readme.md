이 레포지터리는 https://laravel.kr의 소스코드를 관리하기 위한 프로젝트입니다


## Install

```bash
git clone git@github.com:laravelkr/website.git

cd website

composer install
npm ci

php artisan docs:initialize #laravelkr/docs 레포지터리를 다운로드
php artisan docs:update #각 버전별 문서 파일을 다운로드 및 업데이트

```
